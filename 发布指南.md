# GitHub 发布指南

## 发布步骤

### 1. 初始化 Git 仓库（如果还没有）

```bash
cd C:\Users\Administrator\Desktop\ai
git init
```

### 2. 添加远程仓库

```bash
git remote add origin https://github.com/432539/free-sora.git
```

### 3. 添加所有文件

```bash
git add .
```

### 4. 提交更改

```bash
git commit -m "Initial commit: AI内容生成平台 v1.0.0"
```

### 5. 推送到 GitHub

```bash
# 如果仓库已存在，先拉取
git pull origin main --allow-unrelated-histories

# 推送到 main 分支
git push -u origin main

# 或者推送到 master 分支（根据你的仓库设置）
git push -u origin master
```

## 注意事项

1. **确保敏感信息已排除**：
   - `.env` 文件已在 `.gitignore` 中
   - 数据库文件已排除
   - 上传文件目录已排除

2. **检查文件大小**：
   - 确保 `node_modules` 已排除
   - 视频文件（如 `public/anli1.mp4`）如果太大，考虑使用 Git LFS 或移除

3. **首次推送可能需要认证**：
   - 使用 GitHub Personal Access Token
   - 或配置 SSH 密钥

## 后续更新

```bash
git add .
git commit -m "更新说明"
git push origin main
```
