# 设计规范文档

## 界面设计参考

基于提供的参考图片，制定以下设计规范：

### 1. 首页设计

#### 布局结构
- **顶部状态栏**: 时间、信号、WiFi、电池（移动端）
- **用户信息栏**: Logo + 用户名 + ID
- **主Banner**: "一句话生成视频" 大标题 + 示例缩略图
- **功能按钮**: Sora2创作灵感视频（渐变按钮）
- **功能网格**: 2列布局，展示各种AI功能卡片
  - AI生成视频
  - AI生图改图
  - AI复刻视频
  - 视频去水印
  - 图片去水印
  - 其他工具功能
- **底部导航**: 固定底部，5个菜单

#### 视觉元素
- 深色背景 + 渐变按钮
- 功能卡片: 圆角矩形、图标+文字、hover效果
- 图标使用彩色渐变，增强视觉识别

### 2. 作品页面设计

#### 布局结构
- **顶部标签栏**: 
  - 内容类型: 常规视频、短剧作品、生图作品
  - 状态筛选: 全部、生成中、已完成、失败（带数量）
- **VIP横幅**: 分销福利、推广奖励（可选）
- **操作按钮**: 联系客服、常见问题
- **内容区域**: 
  - 空状态: 插画 + "没有数据哦~"
  - 有数据: 网格布局展示作品缩略图
- **提示信息**: "Sora视频制作为您保留24小时，请及时下载。"

#### 交互设计
- 标签切换: 平滑过渡动画
- 作品卡片: 点击查看详情、长按操作菜单
- 下拉刷新、上拉加载更多

### 3. 广场页面设计

#### 布局结构
- **顶部标题**: "案例展示" + 操作按钮
- **内容网格**: 2列布局，展示公开作品
  - 视频缩略图 + 播放图标 + 播放量
  - 文字描述（场景说明）
- **筛选功能**: 按类型、热度、时间排序

#### 视觉元素
- 视频卡片: 圆角、阴影、播放按钮覆盖
- 播放量显示: 左下角，格式化显示（如: 13.67万）

### 4. AI助手页面设计

#### 布局结构
- **对话界面**: 
  - 消息列表（用户消息 + AI回复）
  - 输入框（底部固定）
  - 发送按钮
- **快捷功能**: 
  - 创作建议
  - 使用教程
  - 常见问题

#### 交互设计
- 消息流式输出（打字机效果）
- 支持图片、视频预览
- 历史对话记录

### 5. 我的页面设计

#### 布局结构
- **用户信息区**: 
  - 头像（Logo）
  - 用户名 + ID
- **会员/算力区**: 
  - 视频算力横幅（金色主题）
  - 统计数据: 累计生成、我的算力、高清次数、赠送算力
- **功能列表**: 
  - 清理缓存
  - 我的作品
  - 分销中心
  - 分享海报
  - 卡密兑换
  - 去除水印
  - 常见问题
- **操作按钮**: 联系客服

#### 视觉元素
- 金色用于VIP/会员相关功能
- 列表项: 图标 + 文字 + 右箭头
- 统计数据: 数字突出显示

## 组件设计规范

### 底部导航栏组件

**位置**: 固定底部，z-index: 1000

**样式**:
- 高度: 60px (移动端) / 70px (桌面端)
- 背景: 半透明深色 (#1e293b / 0.95)
- 图标: 24px × 24px
- 文字: 12px，选中状态高亮

**菜单项**:
1. 首页 - 蓝色房子图标
2. 作品 - 绿色对勾图标
3. 广场 - 红色火焰图标
4. AI助手 - 黄色/橙色图标
5. 我的 - 紫色齿轮图标

**交互**:
- 点击切换路由
- 选中状态: 图标和文字高亮，底部指示条

### 生成器组件

**文生图/图生图**:
- 输入区域: 文本输入框（多行）或图片上传
- 参数设置: 
  - 图片尺寸
  - 生成数量
  - 风格选择
  - 高级参数（可选）
- 生成按钮: 渐变背景，加载状态
- 结果展示: 网格布局，支持下载、分享

**文生视频/图生视频**:
- 输入区域: 文本输入框或图片上传
- 参数设置:
  - 视频时长
  - 分辨率
  - 帧率
  - 风格选择
- 生成按钮: 显示预计时间
- 进度显示: 进度条 + 百分比
- 结果展示: 视频播放器，支持下载

### 作品卡片组件

**布局**:
- 缩略图: 16:9 比例，圆角 8px
- 信息层: 标题、创建时间、状态标签
- 操作层: 播放/查看、下载、删除（hover显示）

**状态标识**:
- 生成中: 蓝色，带进度动画
- 已完成: 绿色
- 失败: 红色，显示错误信息

### 媒体播放器组件

**视频播放器**:
- 自定义控制栏
- 全屏支持
- 播放速度调节
- 画质选择（如果有）

**图片查看器**:
- 点击放大
- 缩放、旋转
- 下载功能

## 响应式设计

### 移动端 (< 768px)
- 单列布局为主
- 底部导航固定
- 触摸友好的按钮尺寸（最小44px × 44px）
- 手势支持（滑动、长按）

### 平板 (768px - 1024px)
- 2列布局
- 侧边栏导航（可选）
- 更大的点击区域

### 桌面端 (> 1024px)
- 多列布局（3-4列）
- 侧边栏导航
- 悬停效果
- 键盘快捷键支持

## 动画规范

### 过渡动画
- 页面切换: 300ms ease-in-out
- 组件显示: 200ms ease-out
- 按钮点击: 100ms ease-in

### 加载动画
- 骨架屏: 内容加载时显示
- 进度条: 任务进行时显示
- 旋转图标: 简单加载状态

### 反馈动画
- 按钮点击: 轻微缩放 (scale: 0.98)
- 成功提示: 从上方滑入
- 错误提示: 震动效果

## 颜色系统

### 主色调
```css
--primary: #6366f1;        /* 主蓝色 */
--primary-dark: #4f46e5;   /* 深蓝色 */
--primary-light: #818cf8;  /* 浅蓝色 */

--secondary: #8b5cf6;      /* 紫色 */
--accent: #fbbf24;         /* 金色（VIP） */
```

### 功能色
```css
--success: #10b981;        /* 成功/完成 */
--warning: #f59e0b;        /* 警告/生成中 */
--error: #ef4444;          /* 错误/失败 */
--info: #3b82f6;           /* 信息 */
```

### 中性色
```css
--bg-primary: #0f172a;     /* 主背景 */
--bg-secondary: #1e293b;   /* 次背景 */
--bg-tertiary: #334155;    /* 三级背景 */

--text-primary: #f1f5f9;   /* 主文字 */
--text-secondary: #cbd5e1; /* 次文字 */
--text-tertiary: #94a3b8;  /* 三级文字 */

--border: #475569;         /* 边框 */
```

## 字体规范

### 字号系统
- 超大标题: 32px / 2rem
- 大标题: 24px / 1.5rem
- 标题: 20px / 1.25rem
- 正文: 16px / 1rem
- 小字: 14px / 0.875rem
- 极小字: 12px / 0.75rem

### 字重
- 粗体: 700 (标题)
- 中等: 500 (强调)
- 常规: 400 (正文)

## 间距系统

基于 4px 基础单位:
- xs: 4px
- sm: 8px
- md: 12px
- lg: 16px
- xl: 24px
- 2xl: 32px
- 3xl: 48px

--shadow-lg: 0 10px 15px rgba(0,0,0,0.2);
