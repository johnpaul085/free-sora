## 设计规范总结

### 颜色系统
- **主色**: 蓝紫色渐变 (#6366f1 → #8b5cf6)
- **次要色**: 青色到蓝色 (#06b6d4 → #3b82f6)
- **强调色**: 金黄色 (#fbbf24, #f59e0b)
- **背景**: 黑色 (#000000)
- **文字**: 白色 (#ffffff)
- **边框**: 白色半透明 (rgba(255,255,255,0.2-0.4))

### 字体
- **中文**: PingFang SC, Microsoft YaHei
- **英文**: Inter
- **字号**: text-xs, text-sm, text-base, text-xl, text-2xl, text-3xl

### 间距
- 基础单位: 4px
- 常用间距: gap-2, gap-3, gap-4, gap-6

### 圆角
- 小: rounded-lg (8px)
- 中: rounded-xl (12px)
- 大: rounded-2xl (16px)
- 全圆: rounded-full

### 动画
- 过渡时间: duration-300
- Hover效果: scale-[0.98]
- 卡片hover: hover:border-white/30
